Vix Design.

Vix is a design language.

It helps you design your web apps, It's easy and reliable.

Vix is special, in it's ways. It doesn't define everything for you, like Bootstrap. It only provide the fundamentals, It's up to you to extend them.

Installing

If you use to use an unofficial CDN, put this in your html.

<link rel="stylesheet" type="text/css" href="http://vixdesign.github.io/css/raw/bundles/ui.bundle.css">
        <script type="text/javascript" src="http://vixdesign.github.io/js/bundle.js"></script>

However, it is stronly recommended that you build Vix yourself.

Building

1. Checkout vix's repo on github

git clone https://github.com/vixalien/vixdesign.github.io

2. Install node and stylus.

Install NodeJS for your platform. Use the step-by-step wizard.

Install Stylus: 

npm install -g stylus

3. Build

Got to the root url and type this

stylus css\stylus\bundles -o css\raw\bundles -w -m

This will build all the stylus bundles into the css\raw directory. You can use the omni.bundle.css if you want some minor css for the vix site or ui.bundle.css to only include vix.
